# Anthropic Claude Evaluator

LLM response quality evaluator using Claude as a judge. Supports single evaluation, A/B comparison, and async batch processing.

## Features

- **Single Evaluation**: Rate individual responses on a 0-10 scale
- **A/B Comparison**: Compare two responses to identify the better one
- **Batch Evaluation**: Asynchronously evaluate multiple test cases in parallel

## Installation

```bash
pip install anthropic python-dotenv
```
